# 🛡️ Website Security Audit Tool

A modern web application that analyzes the security configuration of websites by checking HTTPS, SSL certificates, HTTP security headers, response time, and generating a security score.

---

## 📌 Overview

The **Website Security Audit Tool** helps users evaluate the basic security posture of any public website. It provides a simple and user-friendly interface to perform security checks and generate detailed reports.

This project was developed as a **Cyber Security Mini Project** for B.Tech CSE.

---

## ✨ Features

- 🔐 User Registration & Login
- 🌐 Website URL Scanner
- ✅ HTTPS Availability Check
- 🔒 SSL Certificate Validation
- 🛡️ HTTP Security Headers Analysis
- ⚡ Response Time Measurement
- 📊 Security Score (0–100)
- 📜 Scan History
- 📄 PDF Report Generation
- 📱 Responsive Dashboard
- 🌙 Modern UI

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/niiishan07/Website-Security-Audit.git
```

Go to the project directory:

```bash
cd Website-Security-Audit
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

---

## 📖 Usage

1. Register an account.
2. Login to the dashboard.
3. Enter a public website URL.
4. Click **Scan**.
5. View the security analysis.
6. Download the PDF report.
7. Access previous scan history.
